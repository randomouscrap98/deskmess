Desk Mess

A local-only, single-file, lightweight visual project planner and notetaking app inspired by tiddlywiki and featherwiki.

Create cards (like windows) anywhere and link them together to form trees. The current cards are:

  • Markdown page, like the one you see here
  • Chat page, for things like rubberducking
  • Todo list, tracking date of completion

You can also leave yourself links to cards to help with cross-referencing.

NOTES ABOUT SAVING

This is a local-first app. It makes no contact with any server.

You can save a full, working copy of the page exactly as is with the 💾 control at the bottom right. You can even save a copy of this very page! They are unfortunately always a new file on your system due to browser security features. See Alt saving for an alternative save method (still local).

Saved files will never change versions or get styling/scripting updates, because they are always just copies which contain the full app as-is when saving.

If you wish to update, export just the data with 📤, then grab the latest index.html and import with 📥

There is no autosaving by default. Using the normal method, you're responsible for saving and keeping track of local copies.

If this is a bit too much, you can save your files using the browser's local storage by adding ?local=name (replace name with whatever you like). This will:

  • Enable auto saving every minute to the browser
  • Override 💾 to save into the browser

You can use this feature without downloading index.html and it's still all saved locally! Try out the blank page and add your ?local=name to the url. Bookmark the page and you can come back to it, just make sure to use the same browser and don't clear local storage.

Portable version

If neither of those options sound good and you have python installed, try the portable server option. If you've downloaded one of the releases, you can run python3 server.py within the release folder and visit http://localhost:60003 to get a local instance of a server-based desk mess. Saves are stored in a local folder notes next to server.py. You can put the release folder on a flash drive and have a simple way to keep your notes with you.

For more information, see https://feather.wiki/
